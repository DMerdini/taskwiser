# **App Name**: TaskWise

## Core Features:

- Google Authentication: Allow users to sign up and sign in using their Google accounts, streamlining the authentication process and improving user experience.
- Email/Password Authentication: Provide traditional email and password signup and login options as an alternative authentication method.
- Task Management: Enable users to create, update, and delete tasks, including assigning status, team, and comments. Store task information using Firestore.
- Kanban Board: Display tasks on a Kanban board with drag-and-drop functionality for easy status updates.
- Role-based Access Control: Implement role-based access control to restrict access to admin features based on user roles.
- Captcha Verification: Integrate reCAPTCHA on signup forms to prevent bot accounts and improve security.
- AI-Powered Summary: Generate concise summaries of long task descriptions using an AI tool to highlight key information.

## Style Guidelines:

- Primary color: Deep purple (#554971), providing a sophisticated and modern feel.
- Background color: Light grey (#f0f0f0), ensuring content is easily readable and provides good contrast.
- Accent color: Teal (#7acaba) to complement the purple and highlight interactive elements, signaling forward progress and activity.
- Font: 'Inter' sans-serif for a clean, readable, and modern aesthetic suitable for both headlines and body text.
- Use consistent and clear icons from Lucide React for actions and navigation.
- Maintain a clean and organized layout with consistent spacing and padding for a professional look.
- Subtle animations for task transitions to enhance user engagement and experience.
